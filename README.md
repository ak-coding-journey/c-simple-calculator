--C Simple Calculator--

A lightweight calculator program written in C that performs basic arithmetic operations. This project is designed for beginners learning C programming and demonstrates the use of conditional statements, functions, and user input handling.



--Features--

1.Addition, subtraction, multiplication, and division

2.Simple command-line interface

3.Error handling for invalid inputs and division by zero

4.Beginner-friendly code structure



--Usage--

1.Clone the repository:
git clone https://github.com/ak-coding-journey/c-simple-calculator.git

2.Navigate to the project folder:
cd c-simple-calculator

3.Compile the program:
gcc calculator.c -o calculator

4.Run the program:
./calculator



Example:Code-

Enter first number: 10
Enter operator (+, -, *, /): *
Enter second number: 5
Result: 50



--Learning Goals--
1.Practice with conditional statements (if-else, switch)

2.Understand function usage in C

3.Gain confidence with input/output operations
